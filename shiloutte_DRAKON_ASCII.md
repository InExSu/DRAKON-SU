Силуэт ДРАКОН схем Паронджанова наглядно:

+-------+    +-------+              +-------+
|ВЕТКА 1|    |ВЕТКА 2|              |ВЕТКА 3|
+-------+    +-------+              +-------+
    |            |                      |
    v            v                      |
 [Старт]      [Акция]                   |
    |            |                      |
    |            v                      |
    |           / \                     |
    |          /   \                    |
    |          < If >--Нет---           |
    |          \   /        |           |
    |           \ /         |           |
    |            Да         |           |
    |            |          |           |
+-------+    +-------+   +-------+  +-------+
|ВЕТКА 2|    |ВЕТКА 3|   |ВЕТКА 1|  |ВЫХОД  |
+-------+    +-------+   +-------+  +-------+

Mermaid не может отобразить силуэт ДРАКОНа изоморфно.


Ветки горизонтально
```mermaid
flowchart LR
    %% Горизонтальное расположение (LR - Left to Right)
    subgraph ВЕТКА_1["ВЕТКА 1"]
        A1([Старт])
    end

    subgraph ВЕТКА_2["ВЕТКА 2"]
        B1([Акция])
        B2{If}
    end

    subgraph ВЕТКА_3["ВЕТКА 3"]
        C1[Конец]
    end

    %% Основные связи
    A1 --> B1
    B1 --> B2
    B2 -- Да --> C1
    B2 -- Нет --> A1
    C1 --> Exit[ВЫХОД]
```


Ветки вертикально
```mermaid
flowchart TD
    subgraph ВЕТКА 1
        A1([Старт])
    end

    subgraph ВЕТКА 2
        B1([Акция])
        B2{If}
    end

    subgraph ВЕТКА 3
        C1[Конец]
    end

    A1 --> B1
    B1 --> B2

    B2 -- Да --> C1
    B2 -- Нет --> A1

    C1 --> Exit[ВЫХОД]
```
