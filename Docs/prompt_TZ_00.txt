Задача: создать десктоп приложение для работы с диаграммами ДРАКОН

Программа должна считывать код диаграмм из файлов yaml

В коде yaml может быть несколько функций, но отображать нужно будет диаграмму только одной функции.
Визуализатор кода диаграмм должен распологать автоматически иконки и линии между ними.
